### 安装与升级说明

#### Android

- 在系统设置中允许「安装未知来源应用」后，安装 APK 即可。
- 与旧版 Flutter 白守使用**同一包名与签名**，可直接覆盖升级。
- 首次启动后按引导完成设置；若检测到旧版数据，可在应用内执行迁移，恢复伙伴、日记与总结。

#### Windows

- 安装前请先**完全退出**所有白守窗口（包括已安装到 `Program Files\BaiShou` 或其他自定义目录的副本），否则可能出现 `Access denied` 导致安装失败。
- 使用 Inno Setup 四语言安装向导（简体中文 / 繁体中文 / English / 日本語），安装完成后可从开始菜单或桌面快捷方式启动。
- 若从旧版 Flutter 桌面端升级，建议先在新版内完成数据迁移，再卸载旧客户端。

#### Linux

```bash
chmod +x BaiShou-1.0.0.AppImage
./BaiShou-1.0.0.AppImage
```

### 白守是什么

白守不仅仅是一个日记应用，它是为了对抗遗忘而构建的「灵魂容器」。

这是一款运行在本地的、注重隐私的、具有 AI 辅助分析功能的日记与生活记录应用。你可以和 AI 伙伴对话，它能读取你的日记、搜索你的记忆、帮你回顾过去，并通过层级化的 AI 总结（日记 → 周记 → 月报 → 季报 → 年鉴），编织成一部完整的个人史。

---

### 核心功能

- **数据私有**：日记、记忆、对话数据存储在本地（Markdown / SQLite 等），不上传任何服务器。
- **AI 伙伴系统**：创建多个 AI 伙伴，各自拥有独立人格、系统提示词和模型配置；支持 Gemini、OpenAI（DeepSeek / ChatGPT）、Anthropic 等主流模型。
- **RAG 语义记忆**：向量检索 + 全文检索 + 融合排序；日记自动嵌入，Agent 可将重要对话存入记忆库，真正读懂你。
- **智能日记工具**：Agent 可调用日记读写工具；支持一键记忆总结，构建记忆金字塔。
- **网络搜索**：多引擎搜索；部分供应商支持 grounding 搜索。
- **MCP 协议**：标准 SSE 传输，可被外部 AI 客户端调用。
- **多工作区（Vault）**：多个独立工作区，数据完全隔离。
- **弹性备份**：全量快照导出/导入、局域网快传、增量云同步。
- **个性化**：主题切换与多语言（简体中文 / 繁体中文 / English / 日本語）。

### 本版亮点

#### 全新架构

- 桌面端：Electron + React + TypeScript（electron-vite）
- 移动端：Expo / React Native
- 共享核心：`packages/core`、`packages/ai`、`packages/shared`
- 数据库：libSQL / SQLite + Drizzle ORM

#### 从 Flutter 版平滑迁移

- 支持从旧版 Flutter 客户端导入伙伴、会话、日记、总结与头像。
- 自动识别并合并旧版 SQLite 数据库与 Vault 目录结构。
- Android 正式包与旧版共用 upload 证书，用户可无缝覆盖升级。

#### 局域网快传与备份

- 跨端（桌面 ↔ 移动）局域网设备发现与数据传输。
- 移动端支持大包传输进度显示；修复多项连接超时与误报失败问题。
- 全量导出改走外部存储直打 ZIP，快照与归档流程更可靠。

#### 桌面端打包与安装

- Windows 安装包改用 **Inno Setup**（与旧版 Flutter 客户端同款安装器体验）。
- 打包后自动执行运行时依赖校验，避免「能解压但不能启动」的问题。
- Linux 提供 AppImage，开箱即用。

#### 稳定性与体验修复

- 归档导入后正确重建总结管线与 Shadow 索引。
- 修复日记全量扫描路径、附件中文路径、LAN 导入后会话恢复等多项问题。
- 统计面板支持重新扫描；附件清理提示正确显示释放空间。
- 各端应用图标与品牌视觉统一对齐。

### 从 Flutter 版升级（简要步骤）

1. **备份**：建议先在旧版中导出全量快照，或确认本地数据已妥善保存。
2. **安装**：下载并安装本 Release 对应平台的安装包。
3. **迁移**：首次启动按 onboarding 引导操作；检测到旧版数据时，按提示完成迁移。
4. **验证**：检查伙伴列表、近期日记、总结与附件是否正常；确认无误后可卸载旧版 Flutter 客户端。
