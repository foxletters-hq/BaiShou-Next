### 安装与升级说明

#### Android

- 与 v1.0.0 及旧版 Flutter 白守使用**同一包名与签名**，可直接覆盖升级。
- 发版时 `versionCode` 已递增；若提示「无法降级」，请确认安装的是本 Release 最新 APK（详见 [打包须知](https://github.com/Anson-Trio/BaiShou-Next/blob/main/docs/%E6%89%93%E5%8C%85%E9%A1%BB%E7%9F%A5.md) §1.5）。
- 若从 Flutter 旧版或 v1.0.0 升级，可在应用内按引导完成**按板块选择性迁移**。

#### Windows

- 安装前请先**完全退出**所有白守窗口，再运行安装包。
- 从 v1.0.0 升级无需重新迁移数据；若此前未完成旧版迁移，可在设置中继续操作。

### 本版更新摘要

#### 旧版数据迁移与归档恢复

- 支持从 **Flutter 旧版 / 桌面旧版** 按板块选择性迁移（伙伴、日记、总结、配置等）。
- 桌面端提供完整迁移设置页；移动端提供非阻塞引导弹窗。
- 归档导入增强：识别带 manifest 的原版 ZIP、细粒度进度展示、跨平台兼容修复；大包导入 OOM 问题显著改善。
- 迁移完成后提示重启，并阻塞式重建 Shadow 索引。

#### 增量同步（重大改版）

- 从「仅上传 / 仅下载」升级为 **三向增量同步**，支持同步前 **预览计划、确认、重新规划**。
- 删除传播可选择策略；桌面端与移动端确认流程对齐。
- 本地扫描并行化，移动端接入原生 MD5 加速。

#### 日记

- 移动端日记编辑改用 **WebView + CodeMirror 6 Live Preview**，与桌面共用 `shared/diary-codemirror` 层。
- 磁盘格式不变，仍为 `![file](attachment/xxx | width)` Markdown，**无需数据迁移**。
- 新增日记 **标签解析、配色与索引元数据**；附件图片加载与预取优化。
- 同步完成后自动调度受控日记批量嵌入；修复日记未修改却提示保存、未指定尺寸图片预览过大等问题。

#### 聊天与界面体验

- 桌面端与移动端均支持 **自定义聊天背景**（选图、裁剪、模糊、遮罩、透明度）。
- 名称与操作按钮在自定义背景下 **自动反色**；默认应用模糊遮罩。
- 移动端伙伴页侧滑开栏、键盘跟随滚动、聊天气泡交互优化。

#### TTS 语音合成

- 新增 **MiMo 音色复刻 / 设计**，支持参考音频选择与 base64 直读。
- 桌面端与移动端均支持 **分片朗读、逐段连续播放**。
- GPT-SoVITS 本地设置与 IPC 封装修复。

#### 缓存与性能

- 引入统一 **SWR 缓存层** 与 `CacheCoordinator`，伙伴聊天、总结页、日记列表等接入快路径。
- 写路径通过 domain mutation 总线失效缓存，切换工作区后数据刷新更及时。

#### 桌面端 Git 与版本控制

- 集成 **dugite 内置 Git**，版本控制页支持危险操作确认、回滚、暂存等完整流程。

### 问题修复（精选）

#### 工作区与数据一致性

- 修复切换工作区后总结、周记统计、热力图、伙伴列表、会话未刷新的问题。
- 同步按 **活跃工作区作用域** 过滤；启动时自动注册全部工作区，消除同步规划误报。
- 伙伴头像改为全局存储，跨工作区解析；切换头像后聊天界面立即刷新。

#### 时区与日期

- 统一使用 **本地时区** 格式化日期，修复回忆搜索、message_search、prompt 快捷键在 UTC 边界偏移的问题。

#### 移动端专项

- 修复 realme/ColorOS **外部存储权限** 检测与设置跳转。
- 修复 RAG 记忆页在部分机型 **闪退**、伙伴管理页有数据却 **列表空白**、release **签名配置缺失**。
- 导航转场白闪、增量同步弹窗滚动不灵敏、日记编辑器与工具栏重叠等 UI 问题。
- Flutter 旧版迁移后 OOM、归档 ZIP 兼容性。

#### 桌面端专项

- S3 流式下载与并发槽位，规避 Windows 管道错误。
- 工作空间切换持久化、用户档案保存后刷盘、同步后刷新各页数据。
- 旧版日记索引收敛、TTS 与同步类型检查错误。
- 聊天背景样式重复导入、日记模板重置误清伙伴提示词。

#### 数据库与兼容

- 旧库 Agent schema、cache token 列、order_index、PRAGMA 写语句等兼容迁移。
- Drizzle 外壳解包以支持 ATTACH 旧版 SQLite。
