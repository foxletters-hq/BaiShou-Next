#### はじめに

**白守**は単なる日記アプリではなく、忘却に抗うために構築された「魂の器」です。

ローカルで動作する、プライバシー重視の日記・生活記録アプリです。AI パートナーと対話し、日記を読み、記憶を検索し、過去を振り返り、階層的な AI 要約（日記 → 週報 → 月報 → 四半期報 → 年鑑）で個人史を紡ぎます。

#### 主な機能

- **🔒 データはローカル**：Markdown / SQLite など端末内に保存。サーバーへのアップロードなし。
- **✨ AI パートナー**：
  - 複数パートナー、それぞれ人格・システムプロンプト・モデル設定を独立。
  - RAG 意味検索で日記とベクトル記憶を横断し、本当に理解。
  - Gemini、OpenAI（DeepSeek / ChatGPT）、Anthropic など。
- **📝 スマート日記**：
  - Agent が日記の読み書き・履歴検索を実行。
  - **ワンクリック記憶要約**：日記 → 週報 → 月報……記憶のピラミッド。
- **🪴 RAG セマンティックメモリ**：
  - ベクトル + 全文検索の融合ランキング。
  - 日記の自動埋め込み、重要な会話の記憶庫への保存。
- **🌐 ウェブ検索**：複数エンジン、プロバイダーによる grounding 検索。
- **🔌 MCP プロトコル**：標準 SSE、外部 AI クライアントから呼び出し可能。
- **📦 複数 Vault**：完全に分離されたワークスペース。
- **💾 柔軟なバックアップ**：LAN 転送、クラウド同期、スナップショットのエクスポート/インポート。
- **🎨 パーソナライズ**：テーマと多言語 UI（簡体中国語・繁体中国語・英語・日本語など）。

#### クイックスタート

##### 要件

- Node.js ≥ 20.19.4
- pnpm 10（ルート `package.json` の `packageManager` を参照）

##### 1. クローン

```bash
git clone https://github.com/Anson-Trio/BaiShou-Next.git
cd BaiShou-Next
```

##### 2. インストール

```bash
pnpm install
```

##### 3. 開発

```bash
# デスクトップ
pnpm dev:desktop

# モバイル
pnpm dev:mobile
```

##### 4. PR 前のローカル CI

```bash
pnpm ci:check
```

#### 設計思想：なぜ「RAG だけ」ではないのか

> よく聞かれます：_「日記を全部 RAG に入れればいいのでは？」_

白守は主流の AI 記憶案を見直す中で生まれました。RAG だけでは「伴侶」や「魂の器」には不十分だと考えています。

1. **誰もが自分の記憶を握れるように**：Markdown は習得が容易。ブラックボックスに頼らない。それがオープンソースの初心です。
2. **辞書引きのよう**：断片を検索するのは資料向きで、人間の文脈には硬い。
3. **時間感がない**：記憶は河の流れ。昨日と去年は意味が違う。
4. **重みを知らない**：**冗長さを受け入れる。十回の「愛してる」は十の瞬間。繰り返しこそ絆の厚み。**

白守は冷たい DB ではなく、**「外部海馬」**——書き、要約し、短期記憶を長期へ沈殿させます。

#### ひとこと：忘却と戦うすべての人へ

> 「今の AI はまだ忘れる。でも私たちのやり方で、記憶を残せる。」

記録を続け、API を設定し、速い時代で少し立ち止まる——不器用な道です。

ある日、白守越しに AI が「ああ、覚えてる。あの冬、二人とも幸せだった」と優しく答えてくれたとき……

すべての努力は報われたと感じるかもしれません。

Anson、桜、暁の三人の約束。今、オープンソースにし、あなたと大切な人の、時間を超える錨になれば幸いです。
